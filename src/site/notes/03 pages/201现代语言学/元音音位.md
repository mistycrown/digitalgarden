| /a/ | [ɑ]  | 韵尾[-u]/[-ŋ]之前           | 后a，ang、ao |
| /a/ | [ɛ]  | [i-]和[-n]之间             | ian、van   |

| /o/ | [o꜂] | 作复韵母韵腹                  |           |
| /ə/ | [ɤ]  | 在单韵母中                   | 戈         |
| /ə/ | [ə]  | 在鼻韵母中，作轻音节韵腹            | en、eng。的  |
| /e/ | [e]  | 在韵尾[-i]之前               | ei、uei    |
| /e/ | [ɛ]  | 作韵腹，无韵尾时                | ie、ve     |
| /u/ | [u]  | 作韵腹                     |           |
| /u/ | [ɷ]  | 作韵尾                     |           |
| /u/ | [w]  | 作韵头（零声母）                |           |
| /u/ | [ʋ]  | 在[f-]声母后                |           |
| /y/ | [y]  | 作韵腹                     |           |
| /y/ | [ʮ]  | 作韵头（零声母）                |           |
| /ɿ/ | [ɿ]  | 在[ts]/[tsʰ]/[s]后作韵母     |           |
| /ɭ/ | [ɭ]  | 在[tʂ]/[tʂʰ]/[ʂ]/[ʐ]后作韵母 |           |
| /ɚ/ | [ɚ]  | 阳平上声音节                  |           |
| /ɚ/ | [ɐɚ] | 去声音节                    |           |
