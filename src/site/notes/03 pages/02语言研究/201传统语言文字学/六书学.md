# 六书说的局限性 

**表意的模糊性**：在汉字的演变过程中，一些汉字的形体发生了较大变化，使得其原本通过 “六书” 所确定的表意方式变得模糊。
**假借和转注的争议性**：“六书” 中的假借和转注一直存在较大的争议。
